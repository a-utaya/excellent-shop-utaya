# Excellent Shop Utaya Website

建築板金を営む「エクセレントショップ内野屋」の公式Webサイトを、Next.jsでリニューアルするためのプロジェクトです。

現行サイトはタウンページ経由で作成されたものですが、今後は自分たちで情報を更新しやすく、見込みのお客様に安心感が伝わるサイトを目指します。

## Project Goal

- 建築板金、雨どい、屋根まわりの修理・施工について、わかりやすく伝える
- 30代後半からシニア世代までが迷わず読める、落ち着いたWebサイトにする
- 「職人本人が直接対応する」「営業電話なし」「地域密着」の安心感を前面に出す
- Next.jsの学習を兼ねながら、実運用できる品質まで育てる

## Target Users

- 福岡市周辺で屋根、雨どい、外装まわりの修理を検討している方
- 訪問営業や中間業者に不安がある方
- 家の劣化や雨漏りが気になり、信頼できる職人に相談したい方
- スマートフォンで見やすく、電話や問い合わせがしやすいサイトを求める方

## Design Direction

「かっこいい」だけではなく、信頼感・読みやすさ・相談しやすさを重視します。

- 文字は大きめで、行間に余裕を持たせる
- 派手な演出よりも、施工実績と人柄が伝わる構成にする
- スマートフォンでも電話番号や問い合わせ導線を見つけやすくする
- シニア世代にも伝わる、専門用語を抑えた文章にする
- 地域の職人らしい誠実さを感じる色・写真・余白を使う
